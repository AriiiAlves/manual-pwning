# Bem vindo

Este guia foi projetado para ser um manual o mais completo possível acerca de Engenharia Reversa e Pwning. Ele está todo em Português-BR.

Temos 2 tipos de conteúdo aqui: O **For-Theorists** e o **Fast-Food**. Se você é um CTFer descontrolado e não suporta textos grandes, temos um guia prática apenas com o que você deve saber para explorar vulnerabilidades e resolver desafios em CTFs. É bem mais enxuto e provavelmente você não vai entender tudo nos mínimos detalhes, então recomendo saber a teoria para resolver desafios mais difíceis.

Dito isso, bons estudos!

Referências utilizadas:

- [Nightmare (Assembly)](https://guyinatuxedo.github.io/index.html) - Toda a hierarquia de tópicos foi baseada neste material, e inclusive alguns tópicos possuem informações extraídas dele. Sendo assim, enquanto esse guia possui a teoria, o Nightmare possui os binários para você treinar, além de writeups (explicação) de cada um deles. Use como material de apoio.
- [Cybersecurity Notes - Ir0nstone](https://ir0nstone.gitbook.io/notes) - Contém guia prático para binary exploitation com excelentes explicações.
- [Gitbook Ganesh](https://gitbook.ganeshicmc.com/introducao_linux) - Material enxuto, mas aborda muito bem o básico.
- [Poch1ta Introduction to Pwning](https://keksite.in/posts/Introduction-to-pwning-1/) - Ótimo material introdutório para pwning
